# 数据Mock服务
通过ModelHandler.mockData方法可以mock数据

支撑的mock的类型

| 类型名称   | 对应类型  |说明                                               |
| :-------: | :-------: |:------------------------------------------------- |
|  string   | dataType |字符串类型                                         |
|   date    | dataType |日期类型          |
|  boolean  | dataType |布尔类型                                           |
|  number   | dataType |数字串类型                                         |
|   url     | dataType |链接         |
| daterange | dataType |日期范围，   |
|    int    | dataType |整数，                      |
|   enum    | bizType  |枚举                                               |
|   name    | bizType  |人名                                               |
|   email   | bizType  |邮箱                                               |
|   id      | bizType  |自增id                                               |
|   guid    | bizType  |guid                                               |
